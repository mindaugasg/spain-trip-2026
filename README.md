## 📦 What's in This Package

### Main Files
- **`andalusia-complete-guide.html`** - Your complete interactive travel guide (127 KB)
  - 6 main tabs, 20+ sub-sections
  - 12 interactive maps with Leaflet.js
  - 22 external photo URLs + 40 gradient cards
  - 100% functional online, can be made offline

### Documentation Files
- **`PROJECT-CONTEXT.md`** - Project overview, file structure, design features
- **`TECHNICAL-GUIDE.md`** - How to edit and modify the HTML guide
- **`TRIP-DETAILS.md`** - Complete itinerary, bookings, decisions, contacts
- **`DESIGN-SYSTEM.md`** - Colors, typography, spacing conventions
- **`CHANGELOG.md`** - Complete history of all changes made
- **`TODO.md`** - Future improvements and remaining tasks

### Image Embedding Tools
- **`embed_images.py`** - Python script to download and embed all 22 images
- **`embed_images.bat`** - Windows wrapper for easy execution
- **`README-Images.md`** - Instructions for embedding images offline

## 🚀 Quick Start

### View the Guide
1. Open `andalusia-complete-guide.html` in any browser
2. Navigate using tabs at top
3. Click map pins for details
4. Follow external links to book activities

### Edit in Claude Code
1. Open the entire folder in Claude Code
2. Read `TECHNICAL-GUIDE.md` for editing instructions
3. Edit `andalusia-complete-guide.html` directly
4. Refresh browser to see changes

### Make it Offline-Ready
```bash
python3 embed_images.py
```
This creates `andalusia-complete-guide-embedded.html` (2-5 MB) with all images embedded.
